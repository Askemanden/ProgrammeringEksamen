# MEGET SEJT PROJEKT I PROGRAMMERING

## GO

Projektet er et lokalt go multiplayer spil med evaluering af positioner indbygger. Eventuelt vil spillet kunne være online multiplayer og eventuelt indeholde en go bot til single player spil

### Kilder

[Go positionsevaluering](https://webdocs.cs.ualberta.ca/~mmueller/ps/goeval.pdf)

### Keybinds

e = evaluer position
s = skip tur (pas). jf. go reglerne ender spillet efter to pas i streg
esc = åben menu (nødvændig for at forlade spillet)
